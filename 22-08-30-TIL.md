### Visual Studio Code 간단한 사용법



- 동시 수정

> Ctrl + Alt + 아래방향키

- 자동 앞, 뒤 태그 생성

> Alt + w

- 여러줄 자동 완성

~~~html
div{hello}*10
h1+h2+p
<!--id="one"-->
h1#one
<!--class="one"-->
h1.one
<!--h1-h6까지-->
h$*6
~~~

#### section 태그, article 태그

- section 태그
> 하나의 그룹화된 컨텐츠를 포함

- article 태그
> 단독으로 사용 가능
